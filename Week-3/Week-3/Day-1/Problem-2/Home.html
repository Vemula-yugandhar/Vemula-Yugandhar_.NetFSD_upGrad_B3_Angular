1. What is JavaScript?
JavaScript is a scripting language used to create dynamic web pages.
2. What is jQuery?
jQuery is a fast and lightweight JavaScript library for DOM manipulation.
3. What is slideToggle()?
slideToggle() shows or hides elements with a sliding animation.
4. What is DOM?
DOM stands for Document Object Model. It represents HTML as objects.
5. Why use jQuery?
jQuery simplifies JavaScript tasks like event handling and animations.
